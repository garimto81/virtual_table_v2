🚀 사용 가능한 API 엔드포인트

GET /api/spreadsheet/info
스프레드시트 기본 정보 및 메타데이터 조회
GET /api/spreadsheet/health
Google Sheets 연결 상태 상세 확인
GET /api/spreadsheet/preview
Type 시트 미리보기 (기본 10행)
GET /api/spreadsheet/url
브라우저에서 열 수 있는 스프레드시트 URL 생성
GET /api/sheets/read
Type 시트 데이터 읽기 (v2 형식으로 변환)
POST /api/sheets/convert-to-sheet
v2 데이터를 Google Sheets 형식으로 변환
